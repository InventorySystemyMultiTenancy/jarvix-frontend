Jarvis Builder

1. Baixe todos os arquivos desta pasta.
2. Coloque-os em uma pasta local no computador.
3. Instale o Python 3.12 pelo site python.org, se ainda nao tiver.
   Durante a instalacao, marque a opcao "Add python.exe to PATH".
4. Execute gerar_jarvis.bat.
5. O script vai escolher um Python com pip, instalar as dependencias e gerar o app.
6. O executavel Jarvis ficara dentro da pasta dist\Jarvis.

Se der erro, o script salva detalhes em build_jarvis.log na mesma pasta.

Observacao: se o Python instalado vier sem pip, o script tenta corrigir com ensurepip.
Se ainda assim falhar, instale novamente o Python pelo instalador oficial em python.org.
